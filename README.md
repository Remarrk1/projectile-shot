# Лабораторная работа по дисциплине "Компьютерная графика" №2. 

Приложение отображает движение снаряда с заданной скорость и под заданным углом.

### Демонстрация выполнения работы программы 
![Анимация](https://github.com/Remarrk1/projectile-shot/blob/main/gif/shoot.gif)
## Описание задачи
Движение снаряда, который выпущен с начальной скоростью V под углом
a к горизонту. Программа должна циклически запрашивать новые значения ско-
рости V и угла a. Выход по клавише ESC.
## Программа реализована на языке:

| Язык | Интерпретатор/Версия | Среда разработки | 
| ------ | ------ | ------ |
| Python | Python / 3.9 | PyCharm 2022 3.2 |

## Используемые библиотеки:
* tkinter - для создания графического пользовательского интерфейса 
* unittest - для написания и выполнения тестов


## Описание возможностей:
- Ввод скорости полета снаряда;
- Ввод угла полета снаряда;
- Демонстрация полета снаряда с заданной скоростью и под заданным углом;
- Выход из программы при нажатии клавиши Esc.


## Установка:
1. Клонируйте репозиторий
```sh
git clone https://github.com/Remarrk1/projectile-shot
```
2. Установите библиотеку tkinter
```sh
pip install tk
```


## Использование: 
1. Открыть терминал в папке проекта и выполнить следующую команду:
```sh
python main.py
```
2. В открывшемся окне в поле "Скорость" задать скорость полета снаряда, 
в поле "Угол" задать угол полета снаряда в градусах
